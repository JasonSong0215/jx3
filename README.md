# 剑网三衍天宗PVE计算器
## 个人制作维护的小项目

## backend 后端  
python3.7  
django  

## frontend 前端
js  
vue  
vue-element-admin  

## 关键文件目录
### backend 
Judge 技能伤害判定  
jx3 技能封装和输出循环模拟  
pwd 自用密码簿  
Skill 技能基础信息  

## 部署项目
### frontend 
进入frontend 目录  

npm install  

npm install --registry=https://registry.npm.taobao.org  

本地开发 启动项目 npm run dev  

打包项目 npm run build:prod
